Customer Behavior Dashboard
Project Overview

This project presents an interactive Customer Behavior Dashboard built using Microsoft Power BI to analyze customer purchasing patterns, demographic trends, and sales performance across different product categories.

The project combines data extraction, cleaning, analysis, and visualization using tools like MySQL, Python, and Pandas to generate meaningful business insights.

The dashboard helps businesses understand customer engagement, subscription trends, product demand, and revenue distribution.

Key Metrics

The dashboard highlights the following KPIs:

Total Customers: 3.9K

Average Purchase Amount: $59.76

Average Customer Rating: 3.75

These metrics provide a quick overview of customer activity and satisfaction.

Dashboard Insights
Subscription Analysis

27% of customers have subscriptions

73% are non-subscribers

This suggests an opportunity to increase subscription adoption.

Revenue by Category

The revenue analysis shows:

Clothing generates the highest revenue

Accessories contribute significantly

Footwear and Outerwear generate comparatively lower revenue

Sales by Category

Sales volume indicates:

Clothing has the highest demand

Accessories show moderate demand

Footwear and Outerwear have lower sales numbers

Revenue by Age Group

Customer revenue contribution by age group:

Young Adults generate the highest revenue

Middle-aged customers follow closely

Adults and Seniors contribute moderate revenue

Sales by Age Group

Purchasing activity shows:

Young Adults are the most active buyers

Middle-aged customers show strong purchasing behavior

Seniors show steady but lower purchase frequency

Dashboard Filters

The dashboard includes interactive filters for deeper analysis:

Gender

Subscription Status

Product Category

Shipping Type

These filters allow users to explore customer behavior across different segments.

Tools & Technologies Used

MySQL – Data storage and querying

Python – Data processing and analysis

Pandas – Data cleaning and transformation

Microsoft Power BI – Data visualization and dashboard creation

Project Workflow

Data stored and queried using MySQL

Data cleaning and preprocessing performed using Python Pandas

Processed data imported into Power BI

Dashboard created using interactive visualizations and KPIs

Business Value

This dashboard helps organizations:

Understand customer purchasing behavior

Identify high-performing product categories

Analyze demographic buying trends

Improve marketing and subscription strategies

Support data-driven decision making
